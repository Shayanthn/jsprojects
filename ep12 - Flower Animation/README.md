# Episode 12 - Flower Animation

An animated flower built with an HTML canvas and JavaScript. The artwork is generated from looping curve logic, layered petal rotations, and a soft glowing stroke effect.

## Features

- Canvas-based procedural animation
- Rotating petal composition
- Click to restart and clear the animation
- Lightweight HTML, CSS, and JavaScript setup

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Run

Open `index.html` in a browser to view the animation.

## Notes

- The animation starts automatically on load.
- The canvas can be clicked to replay the effect.
